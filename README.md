Usa estos comandos en orden para subir los cambios al repositorio `Propuestas_Generales` usando tu Token más reciente:

```bash
# 1. Reset Total
rm -rf .git
git init
git branch -M main

# 2. Preparar archivos
git add .

# 3. Vincular con Repositorio

# 4. Subida Definitiva
```
